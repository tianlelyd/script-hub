快速开始

欢迎使用 Zed！我们很高兴有你的加入。这是一个快速上手的起点。

下载 Zed

macOS

通过下载页面获取最新的稳定版本。如果你想下载我们的预览版本，可以在它的发布页面找到。首次手动安装后，Zed 会定期检查安装更新。

你也可以通过 Homebrew 安装 Zed 稳定版：

brew install --cask zed

以及 Zed 预览版：

brew install --cask zed@preview

Linux

对于大多数 Linux 用户，最简单的安装方式是通过我们的安装脚本：

curl -f https://zed.dev/install.sh | sh

如果你想帮助我们测试新功能，也可以安装预览版：

curl -f https://zed.dev/install.sh | ZED_CHANNEL=preview sh

这个脚本支持 x86_64 和 AArch64 架构，以及常见的 Linux 发行版：Ubuntu、Arch、Debian、RedHat、CentOS、Fedora 等。

如果 Zed 是通过此安装脚本安装的，可以随时运行命令 zed --uninstall 来卸载。Shell 会提示你是否要保留首选项或删除它们。做出选择后，你应该会看到 Zed 成功卸载的消息。

如果此脚本不能满足你的使用情况，遇到运行 Zed 的问题，或卸载 Zed 时出错，请查看我们的Linux 特定文档。

命令面板

命令面板是访问 Zed 中几乎所有功能的主要方式。它的快捷键是你应该首先熟悉的。打开命令面板：cmd-shift-p|ctrl-shift-p。

试试看！打开命令面板并输入 new file。你应该会看到命令列表被过滤为 workspace: new file。按回车键，你就会得到一个新的缓冲区。

任何时候你看到包含 zed: ... 或 editor: ... 等形式的命令说明，都意味着你需要在命令面板中执行它们。

命令行界面

Zed 有一个命令行界面，在 Linux 上，它应该随发行版的 Zed 包一起提供（二进制文件名可能因发行版而异，为简洁起见，后面将使用 zed）。 对于 macOS，CLI 与编辑器二进制文件在同一个包中，可以通过 cli: install Zed 命令安装到系统中，这将创建一个到 /usr/local/bin/zed 的符号链接。 它也可以从此仓库中的 cli 包从源代码构建。

使用 zed --help 查看完整的功能列表。 主要功能：

  • 打开另一个空的 Zed 窗口：zed

  • 在 Zed 中打开文件或目录：zed /path/to/entry（使用 -n 在新窗口中打开）

  • 从 stdin 读取：ps axf | zed -

  • 在终端中启动 Zed 并显示日志：zed --foreground

  • 卸载 Zed 及其所有相关文件：zed --uninstall

配置 Zed

要打开自定义设置来设置字体、格式设置、特定语言设置等，请使用 cmd-,|ctrl-, 快捷键。

要查看所有可用设置，使用 cmd-shift-p|ctrl-shift-p 打开命令面板并搜索 zed: open default settings。 你也可以在配置 Zed 文档中查看所有设置。

在 Zed 中配置 AI

Zed 在编辑器中以多种方式顺畅地集成了大型语言模型（LLM）。 访问 AI 概览页面了解如何在 Zed 中快速开始使用 LLM。

设置快捷键

要编辑自定义键盘映射并添加或重新映射绑定，你可以使用 cmd-k cmd-s|ctrl-k ctrl-s 来启动 Zed 键盘映射编辑器（zed: open keymap editor），或者直接使用 zed: open keymap 打开你的 Zed 键盘映射 json 文件（~/.config/zed/keymap.json）。

要访问默认键绑定集，使用 cmd-shift-p|ctrl-shift-p 打开命令面板并搜索 "zed: open default keymap"。更多信息请参见快捷键。
