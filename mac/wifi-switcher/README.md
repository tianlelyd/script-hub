# WiFi IP 手动切换工具

手动切换 WiFi 网络的 IP 配置（静态 IP 或 DHCP）。

## 使用方法

```bash
sudo ./wifi_ip_switcher.sh
```

然后选择对应的网络配置：

```
===============================
  WiFi IP 切换工具
===============================

  1) superGod_5G  (192.168.32.114)
  2) mengmeng_5G  (192.168.31.114)
  3) DHCP (自动获取)

  0) 退出

请选择 [0-3]:
```

## 配置说明

如需修改网络配置，编辑脚本开头的 `CONFIGS` 部分。
